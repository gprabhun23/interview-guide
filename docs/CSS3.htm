How do you use Flexbox to create a responsive layout?

  • Flexbox simplifies layout design by arranging items in rows or columns.
  • Key properties: display: flex, justify-content, align-items, and flex-wrap.
  • Example:

css

Copy code

.container {

display: flex;

flex-wrap: wrap;

justify-content: space-around;

align-items: center;

}

.item {

flex: 1 1 200px; /* Grow, shrink, and set base size */

}

What is the difference between relative, absolute, fixed, and sticky positioning in CSS3?

  • Relative: Positioned relative to its normal position.
  • Absolute: Positioned relative to the nearest positioned ancestor.
  • Fixed: Positioned relative to the viewport, stays in place during scroll.
  • Sticky: Switches between relative and fixed based on scroll position.
  • Example for sticky:

css

Copy code

.sticky {

position: sticky;

top: 10px;

}

Explain the concept of CSS Grid and its advantages over Flexbox.

  • CSS Grid is a two-dimensional layout system for rows and columns.
  • Advantages:
    • Handles both rows and columns.
    • More control over alignment and spacing.
    • Simplifies complex layouts.
  • Example:

css

Copy code

.grid {

display: grid;

grid-template-columns: repeat(3, 1fr);

gap: 10px;

}

How does the display property work with values like none, block, inline, inline-block?

  • none: Hides the element, removes it from the layout.
  • block: Takes up the full width and starts on a new line.
  • inline: Stays in line, takes only necessary width.
  • inline-block: Combines inline flow with block-like size control.
  • Example:

css

Copy code

.box {

display: inline-block;

width: 100px;

height: 50px;

}

What is the concept of specificity in CSS and how conflicts are resolved?

  • Specificity determines which CSS rule is applied when multiple rules match an element.
  • Order of specificity: Inline styles > IDs > Classes/Pseudo-classes > Elements/Pseudo-elements.
  • Example:

css

Copy code

/* Low specificity */

div {

color: blue;

}

/* Higher specificity */

.example {

color: red;

}

/* Highest specificity */

#unique {

color: green;

}

Describe the difference between nth-child() and nth-of-type() pseudo-classes.

  • nth-child(): Selects the nth child regardless of type.
  • nth-of-type(): Selects the nth occurrence of a specific type.
  • Example:

css

Copy code

p:nth-child(2) {

color: red; /* Second child, even if not a <p> */

}

p:nth-of-type(2) {

color: blue; /* Second <p> only */

}

How does the z-index property work, and what is its scope?
